This is weather API project

How to get an API key

Go to https://api.openweathermap.org login or sign up and generate API key

https://api.openweathermap.org/data/2.5/weather?q=City want to display data&units=metric&appid=YOUR_API_KEY`

How to setup project

clone the repo from github using git clone command  git clone https://github.com/Jkpanchal/weather-app-itechnosol.git

Do npm install and to start dev server run npm run dev command

Tech stack and dependecy
   React,
   Context API,
   Styled component,
   Tailwind css,
   lucide react
